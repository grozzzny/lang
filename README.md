Language module for Yii2 
==============================

## Installation guide

```bash
$ php composer.phar require grozzzny/lang "dev-master"
```
or composer.json:
```json
"require": {
  "grozzzny/lang": "dev-master"
}
```

Run migrations
```bash
php yii migrate --migrationPath=@vendor/grozzzny/lang/migrations
```
